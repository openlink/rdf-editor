OpenLink Structured Data Editor - Enabling a Read-Write Web of Linked Data
Version: {{version}}

This Editor enables editing of Structured Data documents (using abstract RDF Language) stored in a variety of HTTP accessible documents (Turtle or JSON-LD content formats). Actual document access requires the target document is served from a system that supports at least one of the following HTTP based open standards: Linked Data Platform (LDP), WebDAV, SPARQL 1.1 Update, or the SPARQL Graph Protocol. Get started by creating a brand new document or opening up an existing document.
